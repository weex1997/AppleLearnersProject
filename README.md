# Apple Academy's Learners API

## Overveiw
We create database localy using vapor and postgres and deploy it to global server by using heroku.

### Solution Concept 
Provide Academy learners Achievements and contacts channels to build connections.

### App Statment 
An API that stored all academy's learners Projects and data.

### Feature List:
we have two Routes learners & Projects

- Get all learners data
- Get Ptojects data
- add learner data
- Add Projects data
- update learner data
- update Projects data
- delete learner data
- delete Projects data
